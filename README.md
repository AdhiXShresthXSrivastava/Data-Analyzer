# Data Analyzer (Python)

A simple yet powerful data analysis tool built in a single `main.py` file.  
This project allows users to quickly load datasets (CSV, Excel, JSON), generate statistical summaries, check for missing values, and visualize correlations with heatmaps.

## Features
- Load data from CSV, Excel, or JSON formats
- Generate summary statistics for numeric and categorical columns
- Detect and display missing values
- Plot correlation heatmaps for numeric columns
- Lightweight, runs directly from the command line

## Usage
Run the program and enter your dataset file path (with extension) when prompted:

python main.py

Example input:
Enter the name (or path) of your file: example.csv

## Output
- Prints dataset summary
- Shows missing values count
- Displays correlation heatmap
